# 📊 Lifestyle & Productivity Analytics

## 📌 Project Overview

Analyzed how lifestyle factors such as sleep, screen time, and work hours impact productivity using an end-to-end data analytics workflow.

## 🛠 Tools Used

* Python (Pandas)
* SQL
* Power BI
* Tableau
* Google Colab

## 📊 Key Insights

* Moderate sleep leads to higher productivity
* Increased screen time negatively impacts productivity
* More work hours do not always improve productivity
* Efficiency varies across different lifestyle habits
